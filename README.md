# inventory_and_supply_chain_management_analysis

This project features an interactive Power BI dashboard that provides deep insights into supply chain logistics, warehouse operations, and inventory health. By synthesizing data across regions and product categories, the dashboard enables data-driven decision-making to optimize stock levels and reduce transportation overhead.

# Key Features & Insights

**Warehouse Metrics:** Tracks the percentage of space used vs. total capacity.

**Inventory Health:** Analyzes the average number of days it takes to sell current stock (DSI) and how often inventory is replaced (Turnover Ratio).

**Trend Analysis:** A line chart visualizing unit sales from 2020 to 2024, highlighting the 2022 boom.

**Supply Chain Logistics:** Monitoring Lead Time (manufacturing to delivery) across categories like Electronics, Clothing, and Furniture.

**Order Management:** Breakdown of "Back Orders" to improve customer fulfillment strategies.

# Technical Stack

**Tool:** Power BI Desktop

**Data Source:** CSV/Text files

**Data Cleaning:** Power Query Editor (ETL)

**Analytics:** DAX (Data Analysis Expressions) for custom KPIs.

**Visualization:** Gauge Charts, Clustered Column Charts, Line Charts, and Donut Charts.
